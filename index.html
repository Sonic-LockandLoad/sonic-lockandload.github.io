Installation instructions

Sonic: Lock & Load is a Sonic the Hedgehog-based platforming-focused first-person shooter on the GZDoom source port, i.e. a DOOM semi-total conversion that requires DOOM 2's IWAD.

It's not really based on any kind of Sonic game in particular, instead taking inspiration from the entire franchise.

Sonic: Lock & Load is inspired by the entire Sonic the Hedgehog series of games, Sonic Robo Blast 2, DOOM Eternal, various GZDoom mods and various classic FPS games.

Sonic: Lock & Load is a project by Arsalan Kazmi, free software under the 3-clause BSD license.

Chaos (1.x)

Version 1.4

Future release

Sonic: Lock & Load v1.4 is a major update, bringing new levels, new techniques, refinements to the changes made in 1.3.x and major changes to the gameplay and Sonic: Lock & Load's fundamentals.

How many times have I said "welcome to the next level" anyway?

Really? Only three? Funny, seemed like more.

Levels

  • Green Hill is now Sunset Gardens, and has been made easier in order to ease players into Sonic: Lock & Load.
  • Three more levels are planned.
  • All maps loaded before Sonic: Lock & Load will be inaccessible from the menu. You'll still be able to access them with idclev or the map console command.
  • You can now return to Station Square from Green Hill.
  • Added a new bonus level, "Hire This Level", a level that is just 49152 square map units of absolutely nothing.
  • Score is now counted at the end of a level.

Gameplay

  • The Offhand command has been split into Offhand Attack and Offhand Ability. Offhand Attacks are used for attacks (obviously) and Offhand Abilities are used for movement abilities.
  • Movement has been edited - if you're not pressing any movement keys, then you come to a stop faster than usual, and you preserve more aerial momentum. You can now jump as soon as you touch the ground.
  • Campaign bosses have been removed entirely, though the secret bosses in Station Square are still there.
  • Added a low health warning sound.
  • You now always move at the recommended speed, disregarding the cl_run console variable.
  • Nerfed double jump power.
  • The player will now actually drop rings upon taking damage. These rings will disappear after a while.

Character-specific

Sonic
  • Added the Homing Snap, a technique that allows Sonic to dash towards a target at rapid speed, and be suspended mid-air for a few seconds.
  • Added the Quick Step, which allows Sonic to dodge any incoming attacks. He'll be invincible for the duration of the move.
  • Reworked melee combat. Sonic can now perform sword combos with Caliburn if the player repeatedly presses Fire, and the slam attack is now simply performed with Crouch.
  • Brought back the Soul Surge, now activated by performing the Homing Snap with Caliburn and attacking.
  • The Accelerated Pistol now uses a burst fire instead of being fully automatic, and uses a bit more Chaos Energy.
  • Greatly nerfed Wall Jump.
  • Force Shot can now be done twice in quick succession.
  • Eclipse Blaster Mk. V now fires in a series of short bursts with extended end lag, to incentivise quickswapping.
  • Nerfed Hyper Sonic's speed to be the same as Super Sonic. This also applies to Classic Sonic.
  • Nerfed Boost, Sonic can no longer hold it in the air unless Super and Hyper, and speed is nerfed so you must now build up speed to Boost Jump effectively.
Amy
  • Added the ability to slide.
  • Amy can now use the Piko Piko Hammer to parry incoming attacks, which allows her to reflect projectiles, at the cost of her Chaos Energy.
  • Greatly changed how the Piko Piko Hammer works, requiring the right timing for attacking and Hammer Jumping, and removing Hammer Gliding.
Classic Sonic
  • Added Spin Attack, Insta-Shield and Drop Dash.
  • Spin Dash mechanics have been reworked. It's now activated with simply Fire instead of Crouch+Fire.
  • Can no longer punch.
  • Super/Hyper Sonic can now hover.
  • Drift no longer generates Chaos Energy mid-air.
  • Sonic Thok can no longer be aimed left, right or backwards.
  • Sonic Thok can now be activated by pressing Jump mid-air, like in SRB2.
Shadow
  • Chaos Spear can now be used at any time, as Shadow's Offhand Attack.
  • The Dual Assimilator is now more powerful and a lot more risky to use in a small space.
  • Shadow will be able to Chaos Snap thrice if holding the Chaos Emerald.
  • Shadow is now invincible when performing the Chaos Snap.
  • Added the Sub-Machine Gun, as Shadow's starter weapon.

Miscellaneous

  • Title theme is now Un-gravitify.
  • Updated secret bosses.
  • Fixed title screen not displaying correctly on GNU/Linux systems.
  • Added Reserved Mode, a mode that nerfs player movement to fit regular DOOM levels better.
  • A Discord App ID has been added. I'm unsure as to how Discord RPC works, but it's there.
  • If DOOM2.WAD was found in your GZDoom folder, it will be loaded automatically.
  • The "Hell on Earth" chapter (as well as all other chapters) will now be hidden from the main menu. You can still access these levels with the idclev cheat or the map console command.

More to come! Sonic: Lock & Load v1.4 is still in development.

Version 1.3.3

Sonic: Lock & Load v1.3.3 is a minor update, containing some changes to the game's fundamentals and balancing.

  • Increased Shuriken Crossbow damage output and end frames.
  • Both Eclipse Blasters' knockback has been significantly reduced.
  • Increased Piko Piko Hammer damage output.
  • Chaos Emeralds won't replace soulspheres or megaspheres, instead only replacing the BFG9000.
  • Chaos Energy will now rapidly decrease in Dark or Hyper form instead of deducting 2 at once. This is more of a visual change.
  • The amount of energy you get from capsules is now reduced if you're in Super, Dark or Hyper form.
  • Chaos Rings will now give you extra Chaos Energy if transformed (2 for Super, 4 for Hyper and Dark).
  • Fixed an issue where the Recoil Burst would still cost Chaos Energy when transformed.

Version 1.3.2

Sonic: Lock & Load v1.3.2 is a semi-minor update.

  • Sonic can now wall jump.
  • The option for Slot 4 weapons to replace Slot 2 weapons has been removed.
  • Fixed an issue where completing Green Hill's boss was impossible.
  • Directional inputs for Sonic Thok and Chaos Snap now work with controllers.

Version 1.3.1

Sonic: Lock & Load v1.3.1 is a minor update involving mostly bug fixes.

  • Skill 5 (Blue Streak/Thorned Rose/Fun is infinite/All Fail Shadow) no longer gives enemies double health.
  • Fixed an issue where finishing a level as Shadow would end before the victory jingle played.
  • Fixed an issue where Classic Sonic's mugshot wouldn't change when finishing a level.

Rose (0.8)

Oh boy! The Rose Update! This is probably the greatest addition to the mod in ages.

This update marks the project's name change from 'Sonic the Hedgehog in DOOM' to 'Sonic: Lock & Load'.

This was done:

  • as both a reference to His World from Sonic '06
  • because I didn't want Sonic the Hedgehog in DOOM (or Sonic DOOM) to be confused with Sonic Robo Blast 2, which is what most people think of when they hear "Sonic DOOM" (which didn't work at all, even YouTube thinks S:L&L is SRB2)

Here's the changes:

  • Added Amy Rose.
  • Renamed Sonic's Eclipse Blaster to the "Eclipse Blaster Mk. V"
  • New mugshot sprites for Sonic.
  • Nerfed the Speedy Shotgun.
  • Removed the tutorial.
