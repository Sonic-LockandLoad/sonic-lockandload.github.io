Installing on macOS

Requirements

  • macOS 11 or later
  • The latest GZDoom release (you can get GZDoom from zdoom.org)
  • DOOM II: Hell on Earth's doom2.wad (you can get DOOM II from Steam or GOG)
  • Freedoom: Phase 2's freedoom2.wad if you don't want to use DOOM II

Setting up GZDoom

Open up GZDoom's DMG file and drag and drop the GZDoom application to your Applications folder. Then eject the DMG.

Take your DOOM2.WAD file and put it in your /Users/<username>/Library/Application Support/GZDoom/ folder.

(I strangely remember being able to place it inside /Users/<username>/Documents/GZDoom/, though)

Getting a copy of the game

If you already know how to run GZDoom mods, the process is exactly the same.

Simply grab a PK3 from GitHub releases or git clone the Sonic: Lock & Load repository for the latest development version.

Running the game

Open GZDoom in your Launchpad or from your Applications folder.

You'll see a part that says "command-line arguments" or something similar.

Type in -file path/to/Sonic-LockandLoad-vX.X.X.pk3, where path/to/Sonic-LockandLoad-vX.X.X.pk3 is the path to the PK3 you downloaded or repository folder you cloned.

And that's it!

You're now playing Sonic: Lock & Load. Have fun with it, and don't forget to configure the options in case you want to!

If you have any feedback, open an issue on Sonic: Lock & Load's GitHub.
