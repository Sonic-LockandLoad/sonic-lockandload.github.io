Installation instructions

Requirements

  • Computer running Windows, macOS or Linux, or device running Android (GZDoom isn't available on iOS)
  • The latest GZDoom release (you can get GZDoom from zdoom.org)
  • DOOM II: Hell on Earth's doom2.wad (you can get DOOM II from Steam)
  • Freedoom: Phase 2's freedoom2.wad if you don't want to use DOOM II

If you're on Windows, simply download the v1.4 Demo from the front page and put your respective .wad file in there, then open Sonic-LockandLoad.exe; that's all you need to do and you can ignore the rest of this document.

KIND OF IMPORTANT: I don't own an Android phone or a copy of Delta Touch to test myself, so I don't know for sure Sonic: Lock & Load works well on Android. (It'd also be pretty dependent on your device's power, especially in the Cyber Space Arena.)

Setting up GZDoom

Download and unzip GZDoom's folder (Windows/Mac) or install it through your package manager (Linux) if you haven't already.

Linux users can (and Steam Deck users should) get GZDoom as a Flatpak.

Android users should get Delta Touch from the Google Play Store.

Take your DOOM2.WAD file and put it:

  • Windows – same folder as gzdoom.exe
  • Mac – /Users/<username>/Documents/GZDoom/
  • Linux – ~/.local/share/games/doom
  • Steam Deck – ~/.var/app/org.zdoom.GZDoom/.config/gzdoom
  • Android – Internal Storage/OpenTouch/Delta/

Getting a copy of the game

Recommended: If you're on Mac, Linux or Android and want to play the v1.4 Demo, download it, unzip the folder and extract Sonic-LockandLoad-v1.4-demo.pk3, then continue following the instructions.

If you already know how to run GZDoom mods, the process is exactly the same.

Simply grab a PK3 from GitHub releases or git clone the Sonic: Lock & Load repository for the latest development version.

Running the game

Windows and Linux users can use ZDL to make running the game easy, especially if you want to use any of the DLC.

Mac and Android users can use the built-in launcher if they have more than one IWAD.

For Steam Deck users, GZDeck exists, though I haven't tested it and don't have a Deck to test it with. Make sure you read its instructions, too.

Otherwise, run gzdoom -file Sonic-LockandLoad-vX.X.X.pk3 on the command line.

On Android

UNTESTED!! Again, I don't own an Android phone.

Place Sonic-LockandLoad-vX.X.X.pk3 into Internal Storage/OpenTouch/Delta/Mods/. Then touch the keyboard icon, touch the list icon next to 'Mods and wads' and choose Sonic-LockandLoad-vX.X.X.pk3. Then hit play.

And that's it!

You're now playing Sonic: Lock & Load. Have fun with it, and don't forget to configure the options in case you want to!

Like if you want to turn off that nasty texture filtering. Ew.

If you have any feedback, open an issue on Sonic: Lock & Load's GitHub.
