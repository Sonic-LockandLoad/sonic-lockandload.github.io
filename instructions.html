Installation instructions

Requirements

  • Computer running Windows, macOS or Linux
  • An IWAD for DOOM II: Hell on Earth (you can get DOOM II from Steam)
  • The latest GZDoom release (you can get GZDoom from zdoom.org)

Setting up GZDoom

If you've already set up GZDoom, you can skip this step.

Download and unzip GZDoom's folder (Windows/Mac) or install it through your package manager (Linux) if you haven't already.

Take your DOOM2.WAD file and put it:

  • next to gzdoom.exe (Windows)
  • in /Users/<username>/Documents/GZDoom/ (Mac)
  • in ~/.local/share/games/doom (Linux)

Running the game

Windows and Linux users can use ZDL to make running the game easy.

Mac users can use the built-in launcher if they have more than one IWAD.

Otherwise, run gzdoom -file Sonic-LockandLoad-vX.X.X.pk3 on the command line.

If the above confused you, then all you need to do is open Sonic: Lock & Load's PK3 data file with the GZDoom application.

And that's it!

You're now playing Sonic: Lock & Load. Have fun with it, and don't forget to configure the options in case you want to!

Like if you want to turn off that nasty texture filtering. Ew.

If you have any feedback, open an issue on Sonic: Lock & Load's GitHub.
